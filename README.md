# 🏆 Hackathon 2026

## 📌 Problem Statements

Participants must choose **ONE**:

1. AI Explainability & Transparency
2. Digital Trust & Content Authenticity
3. Next-Gen Automation & Intelligent Agents

---

## 👥 Team Rules

* Team size: **4 members**
* Open to all years
* Cross-department teams allowed
* ⚠️ All members must be from the **same year**

---

## 📅 Duration

**6 Days Hackathon**

---

## 📤 Submission Process

1. Fork this repository

2. Create your team folder inside:

   ```
   submissions/team-name/
   ```

3. Add all required files

4. Push your code

5. Create a Pull Request

---

## 📦 Submission Format (MANDATORY)

Each team must include the following:

### 📁 Folder Structure

```
submissions/team-name/
│
├── README.md
├── report.pdf / report.docx
├── ppt.pptx (optional)
└── demo-video-link.txt
```

---

## 📄 Report / PPT Format

Your report or presentation must include:

### 1. Background

* Explain the problem context

### 2. Objective

* What your solution aims to achieve

### 3. Expected Basic Features

* Key features of your system

### 4. Requirements

* Software / hardware requirements

### 5. Expected Deliverables

* Output of your system

### 6. Tools & Technologies

* Tech stack used

### 7. Resources

* References, datasets, links
